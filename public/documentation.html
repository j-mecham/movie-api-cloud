Film Fanatic

A movie api built by Josh Mecham

The Film Fanatic API provides a place for users to receive information
on movies (directors, actors, genres, etc.) as well as the ability to create
a profile to save data about all their favorites.

URL endpoints and request methods

Functionality Endpoint URL Request Method Query Parameters Request Body Response Data
Creates a new user /users POST None JSON object of new user

Example:
 {
  "Username": String,
  "Password": String,
  "Email": String,
  "Birthday": Date
 } 		JSON object of new user with ID

Example:
 {
  "Username": String,
  "Password": String,
  "Email": String,
  "FavoriteMovies": Array,
  "_id": String,
  "__v": Number
 }
Returns a user by username /users/:Username GET :Username None A JSON object of the specified user
(See above for example)
Creates a new entry in the users Favorite Movies list /users/:Username/movies/:MovieID POST :Username, :MovieID None A JSON object of the user with movieID in FavoriteMovies Array
(See above for example)
Returns a list of all movies /movies GET None None Array of JSON movie objects

Example:
 [
  {
   "Genre": {
    "Name": String,
    "Description": String
   },
   "Director": {
    "Name": String,
    "Bio": String,
    "Birth": String,
    "Death": ""
   },
   "_id": String,
   "Title": String,
   "Description": String,
   "Release_Year": String,
   "ImagePath": String,
   "Actors": Array of Strings,
   "Featured": Boolean
  },
  {
   etc.
  }
 ]
Returns data about a movie by title /movies/:Title GET :Title None JSON object of the movie
(See above for example)
Returns all movies with a specified actor in them /movies/actors/:Name GET :Name None Array of JSON movie objects containing specified actor
(See above for example)
Returns data about a director by name /movies/director/:Name GET :Name None JSON object of the director

Example:
 {
  "Name": String,
  "Bio": String,
  "Birth": String,
  "Death": ""
 }
Returns data about a genre by name /movies/genre/:Name GET :Name None JSON object of the genre

Example:
 {
  "Name": String,
  "Description": String
 }
Updates the user info of the user /users/:id PUT :id JSON object with new info to be updated

Example:
 {
  "Username": String,
  "Password": String,
  "Email": String,
  "Birthday": Date
 } 		JSON object of the updated user

Example:
 {
  "Username": String,
  "Password": String,
  "Email": String,
  "FavoriteMovies": Array,
  "_id": String,
  "__v": Number
 }
Deletes a user by id /users/:Username DELETE :Username None Text saying the user has been deleted
Deletes a movie from the user's favorite movie list by name /users/:Username/movies/:MovieID DELETE :Username, :MovieID None JSON object of user with movie removed from list
(See Above for example)
