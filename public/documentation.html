Film Fanatic

A movie api built by Josh Mecham

The Film Fanatic API provides a place for users to receive information
on movies (directors, actors, genres, etc.) as well as the ability to create
a profile to save data about all their favorites.

URL endpoints and request methods

Functionality Endpoint URL Request Method Query Parameters Request Body Response Data
Creates a new user /users POST None JSON object of new user JSON object of new user with ID
Creates a new entry in the users Favorite Movies list /users/:id/:movieTitle POST :id, :movieTitle None Text saying the movie has been added to the list
Returns a list of all movies /movies GET None None List of JSON movie objects
Returns data about a movie by title /movies/:movieTitle GET :title None JSON object of the movie
Returns data about a director by name /movies/director/:directorName GET :directorName None JSON object of the director
Returns data about a genre by name /movies/genre/:genreName GET :genreName None JSON object of the genre
Updates the username of the user /users/:id PUT :id JSON object with new name as "name" key JSON object the user
Deletes a user by id /users/:id DELETE :id None Text saying the user has been deleted
Deletes a movie from the user's favorite movie list by name /users/:id/:movieTitle DELETE :id, :movieTitle None Text saying the move has been removed
